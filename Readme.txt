## Steps to Initialize and Run the App
1. Navigate into the project folder:
    cd client-management-app

2. Install the dependencies:
    npm install

3. Start the application:
    npm start

## How to Run Storybook for Testing
1. Run the following command to start Storybook:
   
   npm run storybook

2. This will open Storybook in your default browser where you can see and interact with the components.

## What We Have Covered in the App
- **Features**: Client management (add, view, reactivate), client search, client status filtering, and responsive UI.
- **Technologies**: React, MUI, Storybook.
- **Responsive Design**: The app adapts to different screen sizes ensuring usability on mobile, tablet, and desktop devices.
